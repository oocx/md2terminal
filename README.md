# md2terminal

A terminal-based Markdown viewer for the command line, built with .NET 10 and Terminal.Gui v2. Primarily used to view reports generated by [tfplan2md](https://github.com/oocx/tfplan2md).

NOTE: This project is experimental. It is not a full, well maintained project (it may or may not become fully maintained in the future). I created this tool as an experiment in one evening using GitHub Copilot.

## Features

- Full-screen TUI with smooth keyboard and mouse scrolling
- Renders headings, paragraphs, code blocks (with syntax highlighting), tables, blockquotes, lists, and horizontal rules
- Collapsible `<details>`/`<summary>` sections — Tab to navigate, Enter to toggle
- `<span style="color:…">` inline HTML color support
- 17 color themes — cycle with `t` / `T`
- Correct Unicode display width for wide characters and emoji (including VS16 emoji)

## Usage

```sh
# View a file
md2terminal path/to/report.md

# Pipe from stdin
tfplan2md | md2terminal -
```

## Keyboard Shortcuts

| Key | Action |
|---|---|
| `↑` / `↓` | Scroll one line |
| `Page Up` / `Page Down` | Scroll one page |
| `Home` / `End` | Jump to start / end |
| `Tab` / `Shift+Tab` | Focus next / previous section |
| `Enter` | Toggle focused section open/closed |
| `-` | Collapse all sections |
| `=` | Expand all sections |
| `t` / `T` | Next / previous theme |
| `q` / `Q` | Quit |

## Requirements

- .NET 10 SDK

## Building

```sh
git clone https://github.com/oocx/md2terminal
cd md2terminal
dotnet build
```

## Running

```sh
dotnet run -- path/to/file.md
```
